# Mathematics & Cryptography

> *The language underneath reality — made tangible and visible.*

| # | Badge | Achievement |
|---|-------|-------------|
| 1 | The Fractal Mapper | Write a program that renders the Mandelbrot Set with real-time zoom into the boundary. |
| 2 | The Cipher Breaker | Crack a monoalphabetic substitution cipher using only pencil, paper, and frequency analysis. |
| 3 | The Attractor | Graph a chaotic strange attractor (Lorenz, Rossler, or Chen) as an interactive 3D visualization. |
| 4 | The Fourier Analyst | Decompose a complex sound wave into its constituent sine waves using an FFT, then reconstruct the original signal from the components. |
| 5 | The Penrose Tiler | Produce a physical aperiodic tiling — Penrose rhombs or the "Hat" monotile — covering at least one square meter. |
| 6 | The Tensegrity Builder | Construct a physical tensegrity structure that supports meaningful weight through tension alone. |
| 7 | The Origamist | Fold a mathematically precise model (Miura fold, hyperbolic paraboloid, or modular polyhedron) from a single uncut sheet and explain the underlying geometry. |
| 8 | The Enigma | Encrypt and decrypt a message using a Vigenere square, then break one encrypted by someone else using Kasiski examination. |
| 9 | The Topologist | Build a physical model of a non-orientable surface (Klein bottle or Mobius mechanism) or a 4D projection (Tesseract / Schlegel diagram). |
| **FINAL BOSS** | **The Generative Artisan** | **Write a custom mathematical engine that directly drives a hardware output device — pen plotter, laser cutter, or CNC — to produce unique physical generative art where no two outputs are identical.** |
